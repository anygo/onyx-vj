<b>3.0.25: FEATURES:</b>

<ul>
	<li>
		<b>Filters:</b><br><br>
		<ul>
			<li>
				Snap to Tempo implemented, filters can respond to tempo events
				<ul>
					<li>A new TempoFilter class can be extended (onyx.filters.TempoFilter)</li>
					<li>Change the tempo by clicking on the Tempometer in the settings window</li>
				</ul>
			</li>
			<li>
				Ability to add filters to global display
			</li>
			<li>
				Ability to mute a bitmap filter
				<ul><li>Hold down alt and select a filter to mute it</li></ul>
			</li>
		</ul>
	</li>
	<li>
		<b>Transitions</b><br><br>
		<ul>
			<li>Transitions are now implemented.  Change the transition and duration with the transition window (underneath the file-browser)</li>
			<li>Loading Onyx Mix files will use transitions and stagger the loading of the mix</li>
		</ul>
	</li>
	<li>
		<b>Global</b>:<br><br>
		<ul>
			<li>Multiple layers and filters can be saved to an Onyx Mix file.  See examples by browsing into the mix folder.
			To save a mix file, click "save layers" in the settings window (you'll have to paste the output into a text file, and rename it .onx)
			</li>
			<li>
				Visualizer plug-in type implemented
				<ul>
					<li>(load an mp3 file, click custom, then select a visualizer to use)</li>
				</ul>
			</li>
			<li>Rendering performance boost on layers that share a common movieclip or image</li>
		</ul>
	</li>
</ul>

<b>3.0.25: USER INTERFACE CHANGES:</b>

<ul>
	<li>
		<b>Layers:</b><br><br>
		<ul>
			<li>Keys 1-6 now select layers</li>
			<li>Keys Q,W,E select pages within the layers</li>
			<li>Preview layers can dispatch mouse events to custom files by pressing control, and clicking on the layer (see DrawingClip.swf)</li>
			<li>Press control and deleting a layer deletes all layers</li>
			<li>Pressing control while loading a layer saves bitmaps and filters for the newly loaded layer</li>
		</ul>
	</li>
	<li>
		<b>Filters:</b><br><br>
		<ul>
			<li>Filter parameters have now moved into a parameter page (click "filters")</li>
			<li>Mute a filter by pressing alt</li>
			<li>FiltersWindow now has a seperate column for Bitmap Filters and Effect Filters</li>
		</ul>
	</li>
	<li>
		<b>Global:</b><br><br>
		<ul>
			<li>Preview windows now display at real time</li>
			<li>Display window has been created, and you can drop filters onto the global display</li>
		</ul>
	</li>
</ul>

3.0.25: BUG FIXES:
<ul>
	<li>Multiple cameras were not being recognized correctly.  Dual cams are now supported</li>
</ul>

3.0.25: KNOWN ISSUES:
